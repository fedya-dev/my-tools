▸ Как да инсталираш yt-dlp и ffmpeg (разгъни)
1Отвори PowerShell като администратор (Win+X → Terminal (Admin))
2Инсталирай winget пакетите:
winget install yt-dlp.yt-dlp
winget install Gyan.FFmpeg
3Затвори и отвори отново терминала (за да се заредят PATH-овете)
4Провери: yt-dlp --version и ffmpeg -version
5Готово! Върни се тук и свали музика 🎵

Алтернативно (без winget):
— yt-dlp: github.com/yt-dlp/yt-dlp/releases → свали yt-dlp.exe
— ffmpeg: gyan.dev/ffmpeg/builds → release build → извлечи ffmpeg.exe
— Постави и двата .exe в C:\Windows\System32 или в обща папка с PATH
